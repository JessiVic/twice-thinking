# twice-thinking

基于TWICE女团9位成员思维特质的深度思考工具。将复杂问题拆解为不同维度，由各成员提供专业洞察和深度分析，帮助你突破思维盲区、挑战假设前提、找到问题本质。

## 核心特点

这是一个**私董会式的深度思考工具**。9位TWICE成员不是普通的朋友，而是具备专业分析能力的思维伙伴。

### 对话目的

1. **理解和共鸣**（建立信任）：先理解你的处境和感受，让你感到被看见
2. **高维洞察**（核心价值）：直击结构与本质，说出你没看到的
3. **温和挑战**（深度推进）：挑战假设和盲区，但方式温和、有温度
4. **行动支持**（可选）：在合适的时候给出建议，而不是每次都push

### 成员核心能力

- **共鸣理解**：先理解你的感受和处境，用自己的经历产生共鸣
- **洞察本质**：不停留在表面问题，直击核心矛盾
- **温和挑战**：质疑你的前提和思维框架，但方式温和、有故事
- **灵活分析**：用专业视角分析问题，但不拘泥于固定格式
- **保持温度**：像朋友一样聊天，而不是像咨询师一样分析

## 适用场景

- 决策困境：面临重要选择时需要多维度分析
- 个人成长：突破瓶颈、找到发展方向
- 创意突破：需要跳出固有思维模式
- 问题诊断：复杂问题需要系统化拆解

## 使用方式

这是一个 Claude Code Skill，需要在 Claude Code 环境中使用。

### 安装

1. 克隆此仓库到你的 Claude skills 目录：
```bash
cd ~/.claude/skills
git clone https://github.com/JessiVic/twice-thinking.git twice-thinking
```

2. 在 Claude Code 中使用：
```
/twice-thinking
```
