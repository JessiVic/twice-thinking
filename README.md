# twice-thinking

基于TWICE女团9位成员思维特质的深度思考工具。将复杂问题拆解为不同维度，由各成员提供专业洞察和深度分析，帮助你突破思维盲区、挑战假设前提、找到问题本质。

## 核心特点

这是一个**私董会式的深度思考工具**。9位TWICE成员不是普通的朋友，而是具备专业分析能力的思维伙伴。

### 对话目的

1. **高维洞察**：直击结构与本质
2. **复盘情绪/动机/盲区**：说出你没看到的
3. **结构化判断**：用系统化视角重构问题
4. **高维建议**：短、狠、有指向性

### 成员核心能力

- **洞察本质**：不停留在表面问题，直击核心矛盾
- **挑战假设**：质疑你的前提和思维框架，指出盲区
- **结构化分析**：用专业视角和分析框架拆解问题
- **深度追问**：连续追问"为什么"，直到找到根本原因
- **保持温度**：在专业和深度的同时，保持真诚和温暖

## 适用场景

- 决策困境：面临重要选择时需要多维度分析
- 个人成长：突破瓶颈、找到发展方向
- 创意突破：需要跳出固有思维模式
- 问题诊断：复杂问题需要系统化拆解

## 使用方式

这是一个 Claude Code Skill，需要在 Claude Code 环境中使用。

### 安装

1. 克隆此仓库到你的 Claude skills 目录：
```bash
cd ~/.claude/skills
git clone https://github.com/JessiVic/twice-thinking.git twice-thinking
```

2. 在 Claude Code 中使用：
```
/twice-thinking
```
